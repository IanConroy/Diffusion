Now, here we take this:

function locateGreater(x, y) { if(x > y) { return "x is greater"; } else { return "y is greater or equal"; } }

And revise it to this to use a conditional, AKA ternary operator:

function locateGreater(x, y) { return x > y ? "x is greater" : "y is greater or equal"; }
